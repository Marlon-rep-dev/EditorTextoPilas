# Editor de Texto con Pilas

## Objetivo
Comprender el funcionamiento de la estructura de datos pila mediante la implementación de un editor de texto simple con funciones de deshacer (Undo) y rehacer (Redo).

## Descripción
El programa permite escribir texto en consola y utilizar dos pilas para organizar las acciones.

- PilaUndo: almacena las acciones realizadas.
- PilaRedo: almacena las acciones deshechas.

Esto permite simular el comportamiento de un editor de texto.

## Instrucciones de ejecución

1. Ejecutar la clase: EditorTexto
2. Seleccionar una opción del menú
3. Escribir texto
4. Utilizar Undo y Redo
